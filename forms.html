Forms

Use this area to create forms. Each form has a text based code within brackets called a token.

Example Token: [Contact_Form]

Add a form to a page by using the "Insert" menu to insert a form's token within the text. You could also just type the token if you remember it. The form will then automatically show up where the token has been inserted.
